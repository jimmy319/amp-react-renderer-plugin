# Example: Build multiple AMP pages with React + global scoped css

## Quick Start

### Dependencies Installation

```
$ npm i
```

### Development Mode

```
$ npm run dev
```

Visit `http://localhost:3000/page1` and `http://localhost:3000/page2` to see how we handle mutiple pages application.

### Prouction Mode

```
$ npm run build
```

```
$ npm start
```
